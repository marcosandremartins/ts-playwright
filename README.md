<h2 id="started">🚀 Getting started</h2>

<h3>Prerequisites</h3>

- [nodejs](https://nodejs.org/en)
- [playwright](https://playwright.dev/)

<h3>Cloning</h3>

```bash
git clone git@github.com:marcosandremartins/ts-playwright.git
```

<h3>Starting</h3>

```bash
npm install
npx playwright test
```
